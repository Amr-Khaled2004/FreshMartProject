# FreshMart Web Project

FreshMart is a web-based supermarket shopping system where users can browse products, add items to the cart, place orders, and track their orders. It also includes an admin dashboard for managing products, stock, and orders.

## Features

- User registration and login
- Browse supermarket products
- Add products to cart
- Checkout and place orders
- Admin product management
- Stock management
- Order status management
- MongoDB Atlas database integration
- Deployed using Render

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB Atlas
- Render
